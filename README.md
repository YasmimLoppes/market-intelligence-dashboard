# market-intelligence-dashboard
Dashboard interativo para análise de ativos de mercado e inteligência de dados, desenvolvido com Python e Streamlit.

# 📊 Dashboard de Inteligência de Mercado

Este projeto é a camada de visualização de dados do meu ecossistema de Inteligência de Mercado. Ele consome dados em tempo real para gerar insights visuais estratégicos.

## 🎯 Objetivo
Transformar dados brutos em informação visual para facilitar a tomada de decisão. Este dashboard simula uma ferramenta de BI (Business Intelligence) usada para monitorar a saúde de ativos financeiros.

## 🛠️ Tecnologias
* **Python**
* **Streamlit:** Framework para criação de Web Apps de Dados.
* **Plotly:** Gráficos interativos e dinâmicos.
* **Pandas:** Manipulação de dados.

## 🚀 Funcionalidades
* Métricas em tempo real (Preço e Variação).
* Gráficos interativos de barras e setores.
* Filtros dinâmicos por ativo.
* Tabela de dados estruturada.
